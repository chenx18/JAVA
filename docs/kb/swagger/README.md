## 学习重点

- Swagger 不是业务代码，而是接口文档工具
- Spring Boot 通过依赖和自动配置接入 Swagger
- Swagger 主要扫描 `@RestController`、`@GetMapping`、`@PostMapping` 等接口注解
- 常用注解先掌握 `@Tag`、`@Operation`、`@Parameter`、`@Schema`
- Swagger 适合跟项目一起启动，Apifox 适合团队接口管理
