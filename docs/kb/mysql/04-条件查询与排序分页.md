# 04-条件查询与排序分页

真实开发中，很少只查全部数据。

更常见的是：

```text
按 id 查
按名称查
模糊搜索
按时间排序
分页查询
```

这些都离不开 `where`、`order by`、`limit`。

---

## 1. where 条件查询

查询 id 为 1 的数据：

```sql
select * from department where id = 1;
```

查询 name 为 test 的数据：

```sql
select * from department where name = 'test';
```

注意：

```text
数字条件可以不加引号
字符串条件要加单引号
```

---

## 2. and 多条件同时满足

```sql
select * from department where id = 1 and name = 'test';
```

含义：

```text
id 必须等于 1，并且 name 必须等于 test
```

两个条件都满足才会查出来。

---

## 3. or 多条件满足一个即可

```sql
select * from department where id = 1 or name = 'test';
```

含义：

```text
id 等于 1，或者 name 等于 test，满足一个即可
```

---

## 4. like 模糊查询

查询 name 中包含 `dev` 的数据：

```sql
select * from department where name like '%dev%';
```

常见写法：

```text
'dev%'：以 dev 开头
'%dev'：以 dev 结尾
'%dev%'：包含 dev
```

---

## 5. order by 排序

按 id 升序：

```sql
select * from department order by id asc;
```

按 id 降序：

```sql
select * from department order by id desc;
```

含义：

```text
asc：从小到大，升序
desc：从大到小，降序
```

如果不写，默认通常是升序：

```sql
select * from department order by id;
```

---

## 6. limit 限制条数

查询前 10 条：

```sql
select * from department limit 10;
```

跳过前 10 条，再查 10 条：

```sql
select * from department limit 10, 10;
```

含义：

```text
第一个 10：跳过多少条
第二个 10：查询多少条
```

---

## 7. 分页查询

分页公式：

```text
offset = (page - 1) * size
```

第 1 页，每页 10 条：

```sql
select * from department limit 0, 10;
```

第 2 页，每页 10 条：

```sql
select * from department limit 10, 10;
```

第 3 页，每页 10 条：

```sql
select * from department limit 20, 10;
```

---

## 8. 常见组合

按名称模糊查询，并按 id 降序：

```sql
select * from department
where name like '%dev%'
order by id desc;
```

分页查询：

```sql
select * from department
order by id desc
limit 0, 10;
```

条件 + 排序 + 分页：

```sql
select * from department
where name like '%dev%'
order by id desc
limit 0, 10;
```

---

## 9. 当前必须记住

```sql
select * from department where id = 1;
select * from department where name like '%dev%';
select * from department order by id desc;
select * from department limit 0, 10;
```

一句话：

```text
where 管筛选，order by 管排序，limit 管分页。
```
