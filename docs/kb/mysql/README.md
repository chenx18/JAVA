## 学习目标

完成基础篇后，应能：

```text
1. 进入 MySQL 终端
2. 查看数据库和表
3. 创建业务表
4. 插入数据
5. 查询数据
6. 修改数据
7. 删除数据
8. 理解 SQL 和 Java CRUD 的对应关系
9. 理解数据库约束如何作为数据兜底
```
