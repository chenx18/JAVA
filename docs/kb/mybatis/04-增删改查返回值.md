# 04-增删改查返回值

MyBatis 中，不同 SQL 类型通常对应不同返回值。

---

## 1. select 返回数据

查询单条：

```java
@Select("select id, name from department where id = #{id}")
Department findById(int id);
```

查询多条：

```java
@Select("select id, name from department")
List<Department> findAll();
```

规则：

```text
查询单条：返回对象或 null
查询多条：返回 List
```

---

## 2. insert 返回影响行数

```java
@Insert("insert into department (id, name) values (#{id}, #{name})")
int insert(Department department);
```

返回值：

```text
1：插入成功 1 行
```

如果主键重复，通常会抛数据库异常。

---

## 3. update 返回影响行数

```java
@Update("update department set name = #{name} where id = #{id}")
int updateById(Department department);
```

返回值：

```text
1：修改成功
0：没有找到对应 id
```

Service 中可以这样判断：

```java
int rows = mapper.updateById(department);
if (rows == 0) {
  throw new DepartmentNotFoundException(id);
}
```

---

## 4. delete 返回影响行数

```java
@Delete("delete from department where id = #{id}")
int deleteById(int id);
```

返回值：

```text
1：删除成功
0：没有找到对应 id
```

Service 中可以这样判断：

```java
int rows = mapper.deleteById(id);
if (rows == 0) {
  throw new DepartmentNotFoundException(id);
}
```

---

## 5. 推荐规则

```text
select：返回对象或 List
insert：返回 int
update：返回 int
delete：返回 int
```

`int` 表示影响行数。

---

## 6. 一句话总结

```text
查询返回数据，增删改返回影响行数。
```
