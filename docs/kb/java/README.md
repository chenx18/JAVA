## 学习重点

- 类是对象的模板
- 构造函数用于创建对象时初始化字段
- 构造函数可以重载
- 方法也可以重载
- Java 判断重载时看参数列表，不看参数名
- `final` 表示不可再次赋值、不可重写或不可继承
- `public` / `private` 本质是访问控制
- 异常用于表达错误和中断错误流程
