# 01-类对象与构造函数

类、对象、构造函数是 Java 面向对象的基础。

---

## 类是什么

类是模板。

它描述一种对象应该有什么字段、能做什么事情。

```java
public class Department {
  private int id;
  private String name;
}
```

这里的 `Department` 就是一个类。

---

## 对象是什么

对象是根据类创建出来的具体实例。

```java
Department d = new Department();
```

可以理解为：

```text
类：图纸
对象：根据图纸造出来的东西
```

---

## 构造函数是什么

构造函数是创建对象时自动调用的特殊方法。

它通常用于初始化字段。

```java
public Department(int id, String name) {
  this.id = id;
  this.name = name;
}
```

使用：

```java
Department d = new Department(1, "研发部");
```

---

## 构造函数的特点

构造函数有几个特点：

```text
名字必须和类名一样
没有返回值
不能写 void
创建对象时自动调用
```

错误写法：

```java
public void Department(int id, String name) {
}
```

这不是构造函数，而是一个普通方法。

---

## 无参构造函数

无参构造函数没有参数。

```java
public Department() {
}
```

它允许这样创建对象：

```java
Department d = new Department();
```

Spring、MyBatis、Jackson 等框架经常需要无参构造函数来创建对象。

---

## 有参构造函数

有参构造函数可以创建对象时直接传入字段值。

```java
public Department(int id, String name) {
  this.id = id;
  this.name = name;
}
```

使用：

```java
Department d = new Department(1, "研发部");
```

---

## this 是什么

`this` 表示当前对象。

```java
public Department(int id, String name) {
  this.id = id;
  this.name = name;
}
```

这里：

```text
this.id：当前对象的 id 字段
id：构造函数传进来的参数
```

如果字段名和参数名一样，就需要用 `this` 区分。
